# 2026-04-03 重要上下文摘要

## 项目目标
- 多 Agent 交易系统：每个交易员（文章/交易记录）→ 独立 TraderAgent，DataAgent 负责数据，ManagerAgent 负责编排/调度/考核。
- 支持增量文章更新、技能扩展、无 GPU 环境（LLM API）。
- 配置驱动调度，所有参数/阈值均可配置。

## 技术栈
- Python 3.11+（类型注解、依赖库要求）
- FastAPI, Typer CLI, SQLAlchemy, Pydantic, APScheduler, PyYAML, OpenAI/Anthropic API, Playwright, DuckDB, Redis

## 代码结构
- agents/manager_agent, trader_agent, data_agent, base.py
- common/config.py, logger.py, utils.py, exceptions.py
- schemas/contracts.py（Pydantic 通信协议）
- cli/main.py（init-config, run-pre-market, run-after-close, scheduler-start）
- config/app.yaml（配置文件）
- data/processed/phase0/（输出目录）
