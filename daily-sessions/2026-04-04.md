# 2026-04-04 daily session

## 背景与目标
- 目标：多博主/多交易员文章与交易记录 → 形成 Trader persona（优先可执行规则）→ 盘前建议 → 盘后考核复盘 → 反馈闭环。
- 约束：不自动下单；无 GPU；先交互验证再长期运行；第一期 A股/ETF/可转债，路由目标偏收益最大化。

## 关键决策（今天确定）
- 运行形态：独立 Python 应用为核心（CLI + 可选长期调度/服务），Claude Code/Openclaw 作为未来“薄壳”交互入口。
- Persona：同一博主允许多个风格并存（StyleCluster），每个簇一套自洽可执行规则集。
- 盘前建议：基于 MarketState（市场态势）动态选择 Top-1 最优风格 + Top-2 备选，并输出可解释理由。
- 第一阶段 MarketState：先用指数/ETF 日线（例如 510300）做 regime/vol 的规则化分类（解释性优先）。
