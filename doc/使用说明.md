# 使用说明

## 淘股吧 Cookie 配置

第一版淘股吧抓取需要手工提供登录后的 Cookie。

### 获取方法

1. 在浏览器中登录淘股吧
2. 打开开发者工具
3. 进入 `Network`
4. 刷新一个登录后可访问的淘股吧页面
5. 选择任意请求，找到 `Request Headers`
6. 复制 `Cookie` 整段内容

### 推荐配置方式

推荐通过环境变量注入，再在配置文件中引用：

```bash
export TGB_COOKIE='你的完整 Cookie'
```

然后在 `trade-strategy-ai/config/app.yaml` 中配置：

```yaml
crawl:
  auth:
    tgb.cn:
      mode: cookie
      cookie: "${TGB_COOKIE}"
```

也可以直接写入配置文件：

```yaml
crawl:
  auth:
    tgb.cn:
      mode: cookie
      cookie: "你的完整 Cookie"
```

不推荐直接把真实 Cookie 提交到仓库。

### 使用方式

抓取程序会将该 Cookie 作为 HTTP 请求头中的 `Cookie` 使用，用于访问登录后可见的列表页、文章页和评论页。

程序约束：

- 不在业务 JSONL 产物中保存 Cookie
- 日志中应避免输出完整 Cookie
- 当出现未登录、验证码、`403`、`429` 时应终止或退避

### 后续规划

第一版只支持手工 Cookie。

后续会补充：

- 自动登录
- Cookie 自动续期
- 浏览器态回退抓取
